== Description ==
Adds the ability to upload and use layers on reports and set them as defaults.

== Installation ==
1. Copy the entire /addreportlayers/ directory into your /plugins/ directory.
2. Copy the SQL from /sql/defaultcategories.sql into database.
3. Activate the plugin.
