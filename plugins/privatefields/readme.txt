== Description ==
Adds the option to make fields private in a report on the admin side so they can't be seen.

== Installation ==
1. Copy the entire /privatefields/ directory into your /plugins/ directory.
2. Copy the SQL from /sql/privatefields.sql into database.
3. Activate the plugin.
